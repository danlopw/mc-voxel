# Minecraft Voxel

![Grass block icon](icons/icon.png)

Minecraft voxel, internally known as mc-voxel, is an ***Extension*** for [M4KC](https://github.com/sashakoshka/m4kc) that aims to improve some things and
adding new things upon.
